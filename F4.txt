GitHub secrets are encrypted environment variables that you can use to store sensitive information, such as access tokens, API keys, and passwords, that your workflows or actions need to access. You can then use these secrets in your workflows or actions without revealing them in your code.

Here are the steps to use GitHub secrets:

Go to your repository on GitHub and click on the "Settings" tab.
Click on "Secrets" in the left sidebar.
Click on "New repository secret" to create a new secret.
Enter a name for the secret and the value of the secret.
Click on "Add secret" to save the secret.
